О сайте

Данный сайт был создан в ходе выполнения семестровой работы по предмету Web-технологии в информационных системах. Нам было предложено написать сайт на различных платформах, включая графические редакторы. Мною был выбран html и css, в ходе выполнения данной работы мною были использованы средства: Блокнот, встроенный в основные версии Windows редактор, и Notepad++, который является схожим по функционалу, однако слегка удобнее при работе с html файлами.

Основной целью создания данного сайта было изучение WEB-программирования на основе html кода. Изучение различных тегов, форматирование текста в html, создание фреймов, использование и редактирование изображений для дальнейшего использования в коде сайта. Разработка документов, сделанная в этой работе были произведены на ОС Windows 7 Максимальная.
